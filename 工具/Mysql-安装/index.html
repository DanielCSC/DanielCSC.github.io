# 下载

  • 点击这里，去往 Mysql 官网下载
  • 戳我，前往网盘下载，版本为 8.0.22.0 ，提取码：ad3l

# 安装

  1. 打开刚刚下载好的安装包，开始安装 MySQL
  • 选择 I accept 然后点击 next 进入下一步

  1. 这里选择 Developer Default ，然后点击 nex t 进入下一步

  1. 这一步是检查安装条件，直接点击 next 进入下一步就可以了

  1. 这里直接点击 execute 执行就可以了，执行完后点击 next 进入下一步

  1. 继续点击 next

  1. 设置 root密码 然后点击 next 进入下一步。

  1. 一路点击 nex t，并 check 你的 root 密码，MySQL 就成功在你的电脑上安装完成了

# 配置环境变量

  1. 安装完成后进入 MySQL 的安装目录，进入 MySQL Sever 的 bin 目录，复制路径

  1. 右键点击 此电脑 ，选择 属性 ，点击 高级系统设置 ，点击 环境变量 ，点击 Path

  1. 点击 新建 ，将刚刚复制的路径 V 上去

  1. win + R , 输入用户名与刚刚设置的密码，进入数据库 ，搞定
